# **App Name**: The Firsts Interactive Archive

## Core Features:

- Hero Section Layout: CSS Grid layout for the Hero section with 'The Firsts' title, 'About' link, 'Showreel' placeholder, and decorative blocks.
- Infinite Marquee Rows: Display five stacked, infinite-scrolling marquees populated with data from JSON files.  Each row scrolls in alternating directions.
- Data Loading: Load data from 5 different JSON files (The Forerunners, The Trendsetters, The Strangers, The Revivers, The Distinguished) into the marquee rows.
- Card Display: Display 'Name', 'Title', and a truncated 'Lives Summary' on cards within the marquee.
- Hover Interaction: Pause or slow down the marquee row on hover to allow users to view the card.
- Detail Modal: Open a modal displaying the full 'Reason behind the Title', 'Lessons', and 'YouTube Title' when a card is clicked.

## Style Guidelines:

- Primary color: Light blue (#A9B7FF) to provide a modern and artistic feel, inspired by the original image.
- Secondary color: Soft orange (#FFB347) to create a vibrant contrast and highlight key elements, reminiscent of the source.
- Background color: Off-white (#F5F5DC) to offer a clean backdrop, enhancing the visibility of the primary and secondary colors.
- Headline font: 'Playfair Display', serif, for elegant 'The Firsts' branding, mirroring the sophistication of the original design.
- Body font: 'Inter', sans-serif, for clean and readable text in the 'About' section and card details.
- Minimalist icons with a thin stroke, matching the linear style of the referenced UI, for interactive elements like the 'About' link and modal close button.
- Bento-Grid layout for the Hero section, creating a dynamic and engaging visual experience.
- Smooth scrolling animations for the marquees, using Framer Motion to enhance the interactive feel.
